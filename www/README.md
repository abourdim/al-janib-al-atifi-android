# 💗 The Emotional Side of Islam — الجانب العاطفي من الإسلام

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **الجانب العاطفي من الإسلام (The Emotional Side of Islam)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 lesson cards on emotions in Islam with Quran verses and practical actions
- Emotional wisdom section with problem/solution format
- Daily habit tracker with streaks
- Knowledge quiz
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 unique themes, Narrator, Responsive design + PWA-ready

## Tech

Pure HTML + CSS + JS. No frameworks, no dependencies. Just open `index.html`.
